搭建hexo博客

##需要安装的程序以及命令
1.node.js :wget url(软件下载的网址)下载压缩文件
tar -xvf node-v14.15.3-linux-x64.tar.xz: 解压下载的文件 -xvf为tar的参数
rm node-v14.15.3-linux-x64.tar.xz : 删除下载的压缩文件
2.apt install vim : 下载vim

3.将Node.js添加到环境变量中 :
    vim etc/profile /* 打开配置文件*/
    export NODE_HOME=/node-v10.15.3-linux-x64
    export PATH=$PATH:$NODE_HOME/bin 
    export NODE_PATH=$NODE_HOME/lib/node_modules
    source etc/profile /*刷新环境变量*/
    env /*查看系统环境变量命令*/

4.npm install -g cnpm --registry=http://registry.npm.taobao.org : 用淘宝源npm下载cnpm
  cnpm install -g hexo-cli : cnpm全局下载hexo-cli
  npm : 包管理器
5. 在安装node.js的文件夹下新建空文件夹存放本地博客
   mkdir 文件夹名

6. 进入新建的文件夹进行以下操作
   sudo hexo init : 初始化hexo博客
   hexo s : 开始hexo博客，即在本地打开hexo服务
   hexo n "博客名" ： 创建新的博客
   hexo clean 清理一下博客
   hexo g ：生成博客
7. 将博客部署到远端github服务器
   去github上新建仓库 github名 + github.io    
   cnpm install --save hexo deployer-git : 安装github部署插件
   vim _config.yml : 打开配置文件添加以下文本
        type: git
        repo: https://github.com/github名+github.io
        branch: master
8. dexo deployer-git : 部署到远端
9. 打开https://github名+github.io即可查看博客

##可能遇到问题

1. hexo init erro
可能的错误 ：未安装git，安装git就行
2. git远程访问错误
找到host文件
打开host文档添加ip地址（DNS查询自己博客域名的ip地址，任意添加几个）
185.199.110.153 xiaxiaoxiao-web.github.io 

3. git代理错误，重新设置代理
git config --global --unset http.proxy
git config --global --unset https.proxy
